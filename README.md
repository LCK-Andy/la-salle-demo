# la-salle-demo

## Install Git on macOS （在 macOS 上安装 Git）

### Step 1: Install Homebrew （步骤 1：安装 Homebrew）

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Step 2: Install Git using Homebrew （步骤 2：使用 Homebrew 安装 Git）

```bash
brew install git
```
